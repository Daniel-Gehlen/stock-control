# StockControl - Professional Inventory Management System

## 🎯 About

StockControl is a modern, full-stack inventory management system built with cutting-edge technologies and professional design patterns. This project demonstrates advanced software engineering practices, from responsive UI/UX design to robust backend architecture.

## 🚀 Technologies & Techniques

### Frontend
- **React 18** - Modern React with hooks and functional components
- **TypeScript** - Type-safe development with strict typing
- **React Router DOM** - Client-side routing and navigation
- **CSS3 with Custom Properties** - Dynamic theming system with CSS variables
- **Responsive Design** - Mobile-first approach with CSS Grid and Flexbox
- **Context API** - Global state management for themes and user preferences
- **Local Storage** - Persistent user settings and preferences

### Backend
- **ASP.NET Core** - Cross-platform, high-performance web framework
- **Entity Framework Core** - Modern ORM for database operations
- **SQL Server** - Enterprise-grade relational database
- **RESTful API** - Clean, scalable API architecture
- **Swagger/OpenAPI** - Interactive API documentation
- **Dependency Injection** - Loose coupling and testability
- **Repository Pattern** - Data access abstraction layer

### Design Patterns & Architecture
- **Component-Based Architecture** - Reusable, maintainable UI components
- **MVC Pattern** - Model-View-Controller separation
- **DTO Pattern** - Data Transfer Objects for API communication
- **Service Layer Pattern** - Business logic encapsulation
- **Responsive Design** - Adaptive layouts for all screen sizes
- **Dark/Light Theme** - User preference-based theming
- **Accessibility** - WCAG compliant design principles
